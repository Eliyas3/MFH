# Misbah Style Hub

Bespoke tailoring website showcasing premium craftsmanship.

## Features
- **Premium Hero Section**: Modern animations and fashion backgrounds.
- **Masterpieces Portfolio**: High-end gallery of bridal, formal, and ethnic wear.
- **Bespoke Style Designer**: Interactive 3-step configurator for custom tailoring.
- **Premium Design**: Glassmorphism UI, smooth scroll animations, and professional typography.

## Getting Started

Follow these steps to run the project locally:

1. **Navigate to the project directory**
   ```bash
   cd misbah-style-hub-main
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## Technologies Used
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion
- **UI Components**: shadcn-ui
- **Icons**: Lucide React
